1. Window Open-
Here's your cheat sheet for the window & rendering part:

SetConfigFlags(FLAG_WINDOW_RESIZABLE) — called before InitWindow; allows the user to drag-resize the window.
InitWindow(1280, 720, "Neon Trigger") — creates the game window with that width, height (in pixels), and title bar text.
SetTargetFPS(60) — caps the loop at 60 frames/second by sleeping the rest of each 16.6 ms slot; saves CPU/GPU since monitors can't show more.
GetMonitorPosition(1) — returns where monitor #2 begins in the desktop's big coordinate map (monitors count from 0).
SetWindowPosition(x, y) — moves the window's top-left corner to that desktop point; combined with the above = "center the window on the second monitor."
LoadRenderTexture(1280, 720) — creates the invisible fixed-size canvas; the game always draws here, so game code can assume the screen is 1280×720 forever.
SetTextureFilter(..., BILINEAR) — makes the canvas blend pixels smoothly (not blocky) when stretched.
GetScreenWidth() / GetScreenHeight() — the window's current real size; used to compute scale = min(w/1280, h/720) for letterboxing.
WindowShouldClose() — true when the user clicks X; ends the main loop.
GetFrameTime() — how long the last frame took (~0.0166s); this is the dt that makes all movement frame-rate independent.
BeginDrawing() / EndDrawing() — "I'm composing a frame" / "show it now"; the FPS sleep happens inside EndDrawing.
ClearBackground(BLACK) — wipes the previous frame; makes the letterbox bars black.
DrawTexturePro(canvas, src, dest, origin, rotation, tint) — stretches the canvas onto the window; src height is negative to cancel OpenGL's upside-down texture storage.
UnloadRenderTexture / CloseAudioDevice / CloseWindow — cleanup at exit: give borrowed memory back.

Memory trick — group them as the window's life story: setup (SetConfigFlags → InitWindow → SetTargetFPS → position → LoadRenderTexture), every frame (input → Update(dt) → Draw to canvas → scale math → Begin/Clear/DrawTexturePro/End), death (unload + close).




2.Press ENTER to continue-
Game::Game() — src/Game.cpp — constructor; its last line setScreen(Screen::Splash) makes the splash the first screen shown.
setScreen(Screen s) — src/Game.cpp — the only way screens change; sets _screen = s, remembers/restores the menu cursor, manages banner timers and mouse-cursor visibility.
Game::Draw(canvas) — src/Game.cpp — runs every frame; switches on _screen, and while it equals Splash, calls drawSplash().
drawSplash() — src/Game.cpp — paints the backdrop, then draws the blinking prompt only when fmodf(GetTime(), 1.2f) < 0.8f (0.8s on, 0.4s off).
drawMenuBackdrop(dim) — src/Game.cpp — draws temp_bg.png stretched to 1280×720 (fallback: dark gradient if missing); dim=true adds a dark overlay, splash passes false.
GetTime() — raylib (raylib.h), used in drawSplash() — seconds since program start; the clock the blink is computed from.
fmodf(a, b) — C math library (<cmath>), used in drawSplash() — remainder of a÷b; chops endless time into repeating 1.2s cycles.
MeasureText(text, size) — raylib, used in drawSplash() — pixel width the text will have; enables centering via (1280 − w) / 2.
DrawRectangle(x, y, w, h, color) — raylib, used in drawSplash() — the translucent black band under the text so it reads on any photo.
ColorAlpha(BLACK, 0.5f) — raylib, used in drawSplash() — makes a color 50% transparent.
DrawText(text, x, y, size, color) — raylib, used in drawSplash() — actually prints the prompt; drawn after the band, so it sits on top.
handleMenus(), case Screen::Splash — src/Game.cpp — every frame checks IsKeyPressed(KEY_ENTER); on press plays tapSelect() and calls setScreen(Screen::Title).
IsKeyPressed(key) — raylib, used throughout handleMenus() — true only on the frame the key goes down (vs IsKeyDown = true while held); prevents one long press firing 30 times.
tapSelect() — static helper in src/Game.cpp — the confirm click; internally Audio::Play(RK::SND_FOOTSTEP_STOP, 0.8f, 1.4f) from src/Audio.cpp.

Memory story: constructor sets Splash → every frame Draw paints backdrop + (maybe) prompt → clock math decides "maybe" → ENTER press = click sound + state variable changes → splash never drawn again.
